# Terminal V2 Workspace (Isolated)

This folder is a greenfield boundary for the new BagBot Terminal V2.

Rules:
- Do not import or reuse any legacy terminal components, layouts, or themes.
- Build new UI, layout, and charting from scratch under this namespace.
- Backend APIs and bot logic are reused, but UI and visuals are not.
- Keep this area self-contained until the architect authorizes integration.
